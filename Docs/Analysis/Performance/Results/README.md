# 벤치마크 결과

성능 벤치마크 결과를 저장하는 디렉토리입니다.

## 📋 결과 파일 형식

각 벤치마크 결과는 다음 형식으로 저장합니다:

### 파일명 규칙
```
YYYY-MM-DD_[테스트명]_[설명].md
```

예시:
- `2024-11-26_PSO_Caching_Benchmark.md`
- `2024-11-27_CommandList_Recording_Performance.md`

### 결과 파일 템플릿

```markdown
# [테스트명] 벤치마크 결과

**날짜**: YYYY-MM-DD
**환경**:
- CPU: ...
- GPU: ...
- RAM: ...
- OS: ...

## 테스트 시나리오
...

## 결과

### Before
| 항목 | 값 |
|------|-----|
| FPS | ... |
| Frame Time | ... |

### After
| 항목 | 값 |
|------|-----|
| FPS | ... |
| Frame Time | ... |

## 분석
...

## 결론
...
```
