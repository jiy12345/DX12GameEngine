# [시스템/구조 이름]

## 개요
전체적인 구조에 대한 간단한 설명 (1-2 문단)

## 구조 다이어그램

```
┌─────────────────┐
│   Component A   │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│   Component B   │
└─────────────────┘
```

또는 ASCII 아트로 구조를 표현합니다.

## 구성 요소

- **역할**: 이 컴포넌트의 주요 역할
- **책임**: 담당하는 작업들

### Component B
